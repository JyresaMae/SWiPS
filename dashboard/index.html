How SWiPS Works

Real-time pedestrian detection to display actuation · ~117.9ms mean latency

👁️
Step 1

DETECT

YOLOv8n AI detects pedestrians in real time via dual IP cameras

🧠
Step 2

DECIDE

Edge computing on Raspberry Pi 5 determines crossing mode

💡
Step 3

ALERT

Portable monitors display color-coded warnings to drivers

Alert Mode Explanation

Three operational states based on pedestrian detection and obstruction logic

🔴

IDLE

No pedestrians detected

Steady RED 60% brightness
🟢

CROSSING

Pedestrians detected · “CROSS NOW”

Steady GREEN 100% brightness
🔴

OBSTRUCTION

Blocked > 30 seconds · display retains CROSSING · backend logs alert & snapshot

Flashing RED 2 Hz frequency
